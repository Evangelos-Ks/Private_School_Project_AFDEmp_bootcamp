1. Το πρόγραμμα κάνει συσχετισμό.

2. Μπορούμε να κάνουμε και με υπάρχοντα δεδομένα συσχετισμό. π.χ. Ένας μαθητής που υπάρχει ήδη μπορούμε να τον επιλέξουμε από λίστα και να του προσθέσουμε ένα υπάρχον assignment από λίστα ή και νέο assignment το οποίο μόλις το φτιάξουμε θα τοποθετηθεί μαζί με τα υπόλοιπα assignments.

3. Στις ημερομινίες start date και end date υπάρχει και έλεγχος για να μην είναι η start date πριν την end date.

4. Οι ημερομινίες μπορούν να τοποθετηθούν και με αντίθετη φορά από αυτή που ζητείται αλλά πάντα ο μήνας πρέπει να βρίσκεται στη μέση.

5. Τα synthetic data δεν επικοινωνούν με το normal mode. π.χ. Αν δουλεύουμε στο normal mode  και εισάγουμε δεδομένα, αυτά τα δεδομένα θα χαθούν μόλις βγούμε στην επιλογή να διαλέξουμε mode.  Το αντίστοιχο θα συμβεί και με το synthetic data mode. Επομένως όσο είμαστε μέσα στο mode που θέλουμε βλέπουμε όλα τα δεδομένα που έχουμε εισάγει. Με το που βγούμε από αυτό το mode τα δεδομένα που εισάγαμε διαγράφονται και το mode βρίσκεται ξανά στην αρχική κατάσταση.
